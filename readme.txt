== Description ==

Plugin này dành riêng cho khách hàng sử dụng dịch vụ của ThueSMS, giúp quý khách gửi tin nhắn vào số điện thoại của khách hàng khi sử dụng Contact Form 7, NinjaForms hoặc Woocommerce.

= Include =

This plugin includes API of ThueSMS service and has the following functions:

- [THUÊ SIM](https://thuesms.com/)
- [API Get Balance](https://thuesms.com/api/get-balance/)
- [API Send SMS](https://thuesms.com/api/api-send-sms/)
- And more infor here [ThueSMS API DOCS](https://thuesms.com/home-api-v2/)

So this plugin use your ApiKey and your SecretKey on ThueSMS to get balance and send sms.

== Installation ==

Các bước cài đặt thủ công:

1. Trước tiên tải plugin này về và có thư mục tên `ThueSMS`. Hãy tải nó lên `/wp-content/plugins/`.
2. Sau đó vào `Quản lý plugins` và `kích hoạt` plugin `ThueSMS - Gửi tin nhắn sms`.
3. Cơ bản đã xong. Bạn có thể vào `Settings / Cài đặt ThueSMS` trong menu của admin để chọn thêm nhiều tùy chọn khác nhau.
